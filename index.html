GitHub Raw 文件代理

本代理用于访问 GitHub 的 Raw 文件，你可以在这里粘贴 GitHub 仓库的 URL，支持raw、blob链接

加载中...

    


      
示例，可以粘贴以下链接到输入框

        
    
          
  • raw URL: https://raw.githubusercontent.com/kaoqy/Github-Prxoy/refs/heads/main/index.html
    • 
            
  • raw URL(with `refs/heads`):https://github.com/kaoqy/Github-Prxoy/raw/refs/heads/main/index.html
    • 
            
  • blob URL: https://github.com/kaoqy/Github-Prxoy/blob/main/index.html
    • 
                 

     
           
/refs/heads 将会自动移除

        
本代理基于 PHP 开发,  GitHub地址
