欢迎使用 GitHub Raw 内容代理

本代理用于访问 GitHub raw 内容, 你可以通过本代理访问 GitHub 的 raw 文件。

使用方法:

你只需要在代理后面添加github的raw路径即可, 比如要访问一个 html 文件:

  • 原始 URL: https://raw.githubusercontent.com/kaoqy/Github-Prxoy/refs/heads/main/index.html
  • 你的代理 URL: 你的域名/proxy.php/kaoqy/Github-Prxoy/main/index.html(例如: 你的域名/proxy.php/kaoqy/Github-Prxoy/main/index.html) )

/refs/heads 将会自动移除

本代理基于PHP开发, GitHub地址
